# Google-Book-Search
Google-Book-Search is a React based book search application.
With this application, user are able to search for books and save
it to their favorites. Users are also able to view their saved books.

# Technologies Used
* Node.js
* Express.js
* React
* Axios
* Mongoose.js
* Mlab
